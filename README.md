# 📷 Digital Photo Forensics Tool

## 📌 Overview

Digital Photo Forensics Tool is a Flask-based web application that analyzes digital images and extracts important forensic information. It helps users inspect image metadata, generate file hashes, perform Error Level Analysis (ELA), and create a downloadable PDF report.

---

## ✨ Features

- 📤 Upload Image
- 📷 Camera Information
- 📅 Date Taken
- 📍 GPS Coordinates
- 🗺 Google Maps Integration
- 📐 Image Resolution
- 📁 Image Format
- 💾 File Size
- 🔒 MD5 Hash
- 🔐 SHA256 Hash
- 🧪 Error Level Analysis (ELA)
- 📊 ELA Score
- ✅ Genuine / Manipulated Image Detection
- 📄 Download PDF Report

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- Bootstrap 5
- Pillow (PIL)
- ExifRead
- ReportLab

---

## 🚀 How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the Flask server:

```bash
python app.py
```

3. Open your browser:

```
http://127.0.0.1:5000
```
